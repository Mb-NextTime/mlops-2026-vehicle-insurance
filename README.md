# MLOps MVP Конвейер обработки потоковых данных

Программная реализация системы непрерывного машинного обучения на табличных данных. 
Проект разработан в рамках задания курса по MLOps. Система эмулирует работу с потоковыми данными, реализуя полный цикл ML-конвейера без использования сторонних фреймворков (MLFlow, Airflow и т.д.).

## 📚 Документация проекта

## Структура проекта

```text

├── run.py                    # Контроллер CLI (Точка входа)

│

        └── ml_models.py      # ML pipeline, обучение и реестр
```

## Сценарии использования (CLI Интерфейс)

Система управляется через главный скрипт `run.py`. Перед началом работы установите зависимости `pip install -r requirements.txt`.

*   **Очистка системы (сброс к нулю):** `python reset_env.py`
*   **Итерация потока (получение батча + дообучение):** `python run.py -mode update`
*   **Генерация отчета:** `python run.py -mode summary`
*   **Инференс лучшей модели:** `python run.py -mode inference -file data/raw/batch_1.csv`
